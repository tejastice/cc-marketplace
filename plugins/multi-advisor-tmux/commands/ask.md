# Ask

指定したペイン番号のエージェントにメッセージを送信します。

## 引数
- `$1`: ペイン番号（例：1, 2, 3）
- `$2`以降: 送信するメッセージ

## 手順

1. ペイン番号とメッセージを引数から取得
2. 以下のコマンドを**個別に**実行:
   - `tmux send-keys -t <ペイン番号> '<メッセージ>'`
   - `tmux send-keys -t <ペイン番号> C-m`（Enterキー送信）

## 実行例

```bash
# ペイン1にメッセージを送信
tmux send-keys -t 1 'このコードをレビューしてください'
tmux send-keys -t 1 C-m
```

## 重要な注意事項

### Enterキーの送り方
- **Claude Code**: `C-m` を使う
- **Gemini CLI**: `Enter` を使う（C-mは効かない場合がある）

### セキュリティ
- コマンドをチェーン（&&や;）しないこと
- 各コマンドは個別に実行する
