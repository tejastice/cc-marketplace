# Layout Multi

マルチエージェント用の推奨レイアウトを構築します。複数のAIエージェントを並列で動かして比較したい場合に使用します。

## レイアウト構成

```
┌──────────────────────┬──────────────────────┐
│                      │ ペイン1: Claude Code │
│                      ├──────────────────────┤
│ ペイン0: メイン      │ ペイン2: Gemini CLI  │
│                      ├──────────────────────┤
│                      │ ペイン3: Codex       │
└──────────────────────┴──────────────────────┘
```

## 手順

1. 現在tmux内か確認
2. 以下のコマンドを実行してレイアウトを構築:

```bash
# レイアウト構築（ワンライナー版）
tmux split-window -h -p 50 \; split-window -t 1 -v -p 67 \; split-window -t 2 -v -p 50 \; select-pane -t 0
```

3. 各ペインでCLIを起動する案内を表示:

```bash
# ペイン1: Claude Code
tmux send-keys -t 1 'claude' C-m

# ペイン2: Gemini CLI（自動承認モード）
tmux send-keys -t 2 'gemini -y' C-m

# ペイン3: Codex（フルオートモード）
tmux send-keys -t 3 'codex --full-auto' C-m
```

## 注意事項
- 既存のペインがある場合は先に終了させるか、新しいtmuxセッションで実行
- 各CLIは個別に起動コマンドを送信（自動起動はしない）
